# AgroGuide AI

## Offline Smart Crop Recommendation System

### Problem Statement
Farmers often struggle to select the right crop based on soil type, season, and water availability. Many existing solutions require internet connectivity, which is not always available in rural areas.

### Solution
AgroGuide AI is an offline Java application that recommends suitable crops using locally stored agricultural data.

### Features
- Crop Recommendation
- Crop Information
- Crop Rotation Advice
- Recommendation History
- Estimated Yield
- Fertilizer Suggestion

### Technologies Used
- Java
- CSV File Handling
- Object-Oriented Programming

### How to Run

1. Download or clone the repository.
2. Place `AgroGuideAI.java`, `crop_data.csv`, and `crop_rotation.csv` in the same folder.
3. Compile the program:

```bash
javac AgroGuideAI.java
```

4. Run the program:

```bash
java AgroGuideAI
```
